# Security Policy

If you discover a security issue in Lukkey App, do not open a public issue with reproduction steps, secrets, or exploit details.

Use GitHub private vulnerability reporting for this repository if it is enabled.

If private reporting is unavailable, send a responsible disclosure to `support@lukkey.com` and include `Security Disclosure` in the subject line.

Please include:

- affected version or commit
- impact summary
- reproduction steps
- logs or screenshots with secrets removed

We will review the report, confirm impact, and coordinate remediation before public disclosure.
